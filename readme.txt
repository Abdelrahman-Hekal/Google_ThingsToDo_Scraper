This is a Python scraper for Google Things to do (https://www.google.com/travel/things-to-do)

Input:
Google sheet directory for the destinations to be used as inputs in the seach in things to do
Google drive directory for the output to be saved
all the previous inputs must be shared with edit permissions

Output:
Google docs for the information with images for the scraped destinations

Other Info:
THe scraper is using Google drive, Docs and Sheets API to process the inputs and generate the outputs
